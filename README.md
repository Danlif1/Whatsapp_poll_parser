# Whatsapp_poll_parser
Transferring polls from Whatsapp to google sheets

Before running the code don't forget to create a file called .env

It should hold the variables: zip_name, chat_name

For example:
```
zip_name="abc.zip"
chat_name="our awesome chat"
```

## To run the code do:
```
./run.sh
```

If it doesn't work do:
```
chmod 777 ./run.sh
```
